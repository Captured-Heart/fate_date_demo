## Getting Started

After cloning,

### FVM setup

it runs on flutter version: flutter: ">=3.22.1", dart: ">=3.3.4 <4.0.0",

Install `fvm` if not already installed.

```bash
dart pub global activate fvm
```

Install the local `flutter` version.

```bash
fvm install
```
